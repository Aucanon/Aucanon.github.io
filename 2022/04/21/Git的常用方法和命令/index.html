Git的常用方法和命令

2022-04-21

.gitignore文件

1.#忽略所有.a文件
*.a
2.#跟踪index.html（即使前面用*.html忽略了所有html文件）
!index.html
3.#忽略当前目录下的index.css文件
/index.css
4.#忽略任何目录下名为index.html的文件夹
index.html/
5.#忽略 lib/1.txt(但不忽略 lib/dir/1.txt)
lib/*.txt
6.#忽略 doc/目录及其所有子目录下的.docx文件
doc/**/*.docx

分支

查看分支

git branch

基于当前分支创建 一个新分支

git branch 分支名

切换到某一分支上

git checkout 分支名

创建，并切换到新的分支上

git checkout -b

将分支合并到master

git merge 分支名

删除分支

git branch -d 分支名

仓库

查看远程仓库中所有分值列表信息

git remote show

从远程仓库把对应的远程分支下载到本地仓库，并把下载的本地分支进行重命名

git checkout -b 本地分支名称 远程仓库名称/远程分支名称

从远程仓库拉取当前分支最新的代码

git pull

删除远程仓库中指定名称的分支

git push 远程仓库名 --delete 远程分支名

移除仓库中的文件 但保留工作区的该文件

git rm --cached 文件名

同时移除工作区和仓库内的文件

git rm -f 文件名

从远程仓库克隆到本地

git clone "url"

回滚

撤销修改

git checkout --文件名

从暂存区内移除

git reset HEAD 要移除的文件名或者全部（.

查看最近两次提交记录

git log -2

在一行上显示提交历史

git log --pretty=oneline

回退到某一ID的版本

git reset --hard <CommitID>

在旧版本中使用git log --pretty=oneline命令 查看命令操作的历史

git reflog --pretty=oneline
