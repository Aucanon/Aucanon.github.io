Vue--v-指令

2021-08-04

v-指令

1.v-text

v-text指令的作用是设置内容的标签，设置文本。数据定义在data中。

<body>
    <div id="app">
        <h2 v-text="message"></h2>
        <h2 v-text="message2"></h2>
    </div>
    <script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
    <script>
        var app = new Vue({
            el: "#app",
            data: {
                message: "learn",
                message2: "hello"
            }
        })
    </script>

</body>

2.v-html

v-html指令的作用是设置innerHTML标签。与v-text不同于v-text只解析为文本。

<body>
    <div id="app">
        <p v-html="content"></p>
    </div>
    <script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
    <script>
        var app = new Vue({
            el: "#app",
            data: {
                content: "<a href='https://www.csdn.net/'>CSDN</a>"
            }
        })
    </script>
</body>

3.v-on

事件绑定用于监听DOM事件，绑定的方法定义在methods中。v-on还可以简写为@。

<body>
    <div id="app">
        <input type="button" value="v-on指令" v-on:click="one">
        <input type="button" value="v-on简写" @dblclick="two">
    </div>
    <script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
    <script>
        var app = new Vue({
            el: "#app",
            methods: {
                one: function() {
                    alert("单击了一下");
                },
                two: function() {
                    alert("双击666");
                }
            },
        })
    </script>
</body>

4.v-if/v-show

在指令条件表达式返回值为true是进行渲染

v-show
通过控制样式的display属性，渲染的元素始终保留在DOM中。如果运行的条件很少改变可以选择使用v-show指令。

<body>
    <div id="app">
        <h2 v-show="isShow">show hello</h2>
    </div>
    <script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
    <script>
        var app = new Vue({
            el: "#app",
            data: {
                isShow: true
            }

        })
    </script>
</body>

v-if
会根据条件来决定条件块内监听器和组件的销毁和重建。

<body>
    <div id="app">
        <h2 v-if="temp === 'a'">看见了1</h2>
        <h2 v-else-if="temp === 'b'">看见了2</h2>
        <h2 v-else="temp === 'c'">看见了3</h2>
        <input type="button" value="next" @click="change1" @dblclick="change2">
    </div>
    <script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
    <script>
        var app = new Vue({
            el: "#app",
            data: {
                temp: "a"
            },
            methods: {
                change1: function() {
                    this.temp = 'b';
                },
                change2: function() {
                    this.temp = 'c';
                }
            },

        })
    </script>
</body>

5.v-bind

元素绑定，操作元素属性的指令。

v-bind:class
根据条件执行class列表内容，来动态的操作class。

<body>
    <div id="app">
        <div v-bind:class="{active:isActive}" style="width:200px;height: 200px;text-align: center;line-height: 200px;">v-bind</div>
        <!-- <div class="static" v-bind:class="active:isActive,'text-danger" :hasError></div>
        <div class="static active"></div> -->
    </div>
    <script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
    <script>
        var app = new Vue({
            el: "#app",
            data: {
                isActive: true,
                // hasError: false
            },
        })
    </script>
    <style>
        .active {
            background-color: #ff0000;
        }
    </style>
</body>

v-bind:style
操作元素的内联样式

<body>
    <div id="app">
        <div v-bind:style="isStyle">v-bind</div>
    </div>
    <script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
    <script>
        var app = new Vue({
            el: "#app",
            data: {
                isStyle: {
                    height: '100px',
                    width: '100px',
                    fontsize: '20px',
                    color: 'red',
                    background: 'black',
                }
            },
        })
    </script>
</body>

6.v-for

响应式的根据数据生成列表结构

<body>
    <div id="app">
        <div>v-for
            <ul>
                <li v-for="item in arr">
                    {{ item }}
                </li>
            </ul>
            <h2 v-for="(item,index) in str">
                {{ acc }} - {{ index }} - {{ item }}
            </h2>
        </div>
    </div>
    <script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
    <script>
        var app = new Vue({
            el: "#app",
            data: {
                arr: ["a", "b", "c", "d"],
                str: ["one", "two", "tree"],
                acc: "time"
            },
        })
    </script>
</body>

7.v-model

表单元素双向绑定，绑定的数据与表单元素值相关联。

<body>
    <div id="app">
        v-model
        <input type="text" v-model="message">
        <h2> {{ message }} </h2>
    </div>
    </div>
    <script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
    <script>
        var app = new Vue({
            el: "#app",
            data: {
                message: "123"
            }
        })
    </script>
</body>

总结

在底层的实现上，Vue 将模板编译成虚拟 DOM 渲染函数。结合响应系统，Vue 能够智能地计算出最少需要重新渲染多少组件，并把 DOM 操作次数减到最少。–就是好用就完了。
